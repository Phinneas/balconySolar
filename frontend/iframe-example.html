About This Example

This page demonstrates how to embed the Balcony Solar Legal State Checker on your website using an iframe. The checker is fully functional and self-contained - it doesn't require any additional dependencies from the parent page.

Features:

  • State selection with dropdown
  • Legal status display with visual indicators
  • Regulatory details and requirements
  • Official resource links
  • Shareable URLs with copy-to-clipboard
  • Responsive design for all devices
  • Print-friendly formatting

Embedding Examples

Basic Embedding

Simple iframe with full width and fixed height:

<iframe src="https://checker.solarcurrents.com" width="100%" height="800" frameborder="0" allow="clipboard-write" title="Balcony Solar Legal State Checker" ></iframe>

Pre-selected State

Embed with a specific state pre-selected (e.g., California):

<iframe src="https://checker.solarcurrents.com?state=ca" width="100%" height="800" frameborder="0" allow="clipboard-write" title="Balcony Solar Legal State Checker" ></iframe>

Responsive Container

Embed with a responsive container and custom styling:

<div style="max-width: 900px; margin: 2rem auto;"> <iframe src="https://checker.solarcurrents.com" width="100%" height="800" frameborder="0" allow="clipboard-write" title="Balcony Solar Legal State Checker" style="border: 1px solid #ddd; border-radius: 8px;" ></iframe> </div>

Fixed Size

Embed with fixed width and height:

<iframe src="https://checker.solarcurrents.com" width="600" height="800" frameborder="0" allow="clipboard-write" title="Balcony Solar Legal State Checker" ></iframe>
